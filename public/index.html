How to export from SketchUp Free →

Exporting from SketchUp Free → GLB

Easiest path — STL (no conversion needed)
Step 1 In SketchUp Free, click the folder icon → Export → STL
Step 2 Upload the .stl file here using the Upload 3D Model button. Done.
Note: STL shows geometry only (no colours or textures). For a coloured model, use the GLB path below.
Coloured model — GLB via online converter
Step 1 In SketchUp Free, click the folder icon → Download → SKP
Step 2 Go to Convertio SKP→GLB (free, no sign-in needed) and convert the file
Step 3 Upload the .glb file here using the Upload 3D Model button
Tip: keep your model under 50 MB for best performance in the browser.
